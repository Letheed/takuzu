# Takuzu

A library and a binary for solving Takuzu (a.k.a. Binairo) number puzzles.

## Documentation

See the [documentation][docs] and the [example grids][grids].

## Usage

To use the solver:

```bash
cargo install takuzu
takuzu [FILE]...
```
